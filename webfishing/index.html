Detailed Guides: (With images!)

  • Install Mods via Modpack
  • Install Mods with Mod Manager (Fall Back)

1. Download Hook, Line, & Sinker & Install it

First step is very simple Download the Mod Loading app: Hook, Line, & Sinker from the button below! Then when windows throws the "protected your PC" window simply press the "more info" and then select: "Run anyways".

Download Hook, Line, & Sinker

2. Setup HLS via the settings page

Click on the HLS Setup tab & link your Webfisher install, verify the installation if necessary, if necessary install .NET(you can skip but if the installing fails then download it), & finally install GDWeave by clicking the button.

3. Install mods!

Click back to the mod manager and install mods by selecting them and then clicking on "Install"

4. Install 3rd party mods!

If you're my friend then you have access to a private mod that must be installed via the "Import ZIP" button. Simply click on it and select the Zip file I have sent over!

5. That's it! Time to play!

Once you got the mods you want start the game! Modded games can join vanilla and vanilla players can join modded games!(For the most part!) Have fun!
